## 🛠️ Stack Tecnológico y Hardware

**Software:**
* **Lenguajes:** C++ (Firmware), Python (Backend)
* **Frameworks:** Arduino Core, Flask
* **Control de Versiones:** Git & GitHub

**Hardware Principal:**
* Placas de desarrollo ESP32 / Arduino Mega
* Sensores ultrasónicos y módulos DHT22
* Protoboards, relés y cableado estructurado
